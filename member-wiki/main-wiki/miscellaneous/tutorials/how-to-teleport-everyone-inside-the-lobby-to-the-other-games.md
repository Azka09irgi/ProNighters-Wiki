# How to Teleport Everyone inside ProNighters Lobby to the Other Games

{% hint style="info" %}
You need to be a Host to use this feature!
{% endhint %}

{% hint style="warning" %}
You'll need to request access to this feature before using it.
{% endhint %}

### Step 1 – Join the ProNighters Lobby

First, join the ProNighters Lobby. You can access it from the Lobby page.

<figure><img src="../../../.gitbook/assets/image (22).png" alt=""><figcaption><p>The ProNighters Lobby page on Roblox</p></figcaption></figure>

### Step 2 – Locate the teleport GUI

Click the GUI labeled "Open Teleports." If you do not see it, you do not currently have access to the Teleport button.

<figure><img src="../../../.gitbook/assets/image (23).png" alt=""><figcaption><p>The "Open Teleports" GUI</p></figcaption></figure>

### Step 3 – Get the Place ID of the experience

Find the Place ID for the experience by checking your browser's address bar. For example, Piggy has the Place ID "4623386862."

<figure><img src="../../../.gitbook/assets/image (24).png" alt=""><figcaption><p>Example of the Place ID</p></figcaption></figure>

### Step 4 – Paste the Place ID into the GUI

Once you have the Place ID, paste it into the GUI and click "Teleport."

<figure><img src="../../../.gitbook/assets/image (25).png" alt=""><figcaption><p>Teleport GUI</p></figcaption></figure>

### Step 5 – Finalize

Congrats! You just learnt how to teleport everyone inside ProNighters Lobby to the other games! If you need further assistance, you can ask our hosts!
